Superclass: Beta-hairpin exposed by an alpha/beta-scaffold domains

Transcription factors in this superclass possess a DNA-binding domain (DBD) structured as an alpha/beta scaffold, with a beta-hairpin inserting into the DNA major groove as the primary DNA-contacting element. In TFClass, this superclass originally included only the GCM domain factors class, represented by a single mammal-specific family (the GCM family).

We expanded it by adding five plant-specific families: NAC, VOZ, CAMTA, WRKY, and FRS/FRF. Additionally, we incorporated the AP2/EREBP class, which includes proteins with the AP2/ERF DBD—an element absent in mammals.
