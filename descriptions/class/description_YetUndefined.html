Superclass: Yet Undefined DNA binding domains

This superclass includes transcription factors (TFs) with functionally well-characterized DNA-binding domains (DBDs), but which still lack structural data for definitive classification. In TFClass, it encompasses four established classes, along with an ‘uncharacterized’ class that includes six families. Among these, plants feature NFX1-like proteins, whose transcription factor activity is yet to be firmly confirmed.

We expanded this superclass by adding six plant-specific classes corresponding to TF families with well-established activity, but for which key structural details regarding the DBD/DNA interaction are still lacking.
