Superclass: Helix-turn-helix domains



Class: LEAFY

The LEAFY (LFY) class is plant-specific. The crystal structure of Arabidopsis LFY DBD (PDB: 2VY1, 2VY2) and Physcomitrium patens LFY (PDB: 4BHK) reveals an HTH motif embedded in a seven-alpha-helix fold, plus a loop that contacts the DNA minor groove. LFY binds DNA as a dimer and can form higher-order assemblies via a SAM oligomerization domain [src] [src].
